Техническое задание для TODO приложения

Описание проекта:

Разработать простое клиентское TODO приложение, позволяющее пользователю создавать, просматривать, редактировать и удалять задачи.

Функциональные требования:

1. Пользователь должен иметь возможность создавать новые задачи с указанием заголовка и описания.
2. Пользователь должен иметь возможность просматривать список всех задач.
3. Пользователь должен иметь возможность редактировать задачу, включая заголовок и описание.
4. Пользователь должен иметь возможность отмечать задачу как выполненную или невыполненную.
5. Пользователь должен иметь возможность удалять задачу.
6. Все изменения должны быть сохранены в базе данных или локальном хранилище.

Технические требования:

1. Приложение должно быть разработано с использованием React.
2. Использование UI-библиотеки Material-UI обязательно.
3. Для взаимодействия с сервером необходимо использовать библиотеку Axios или аналог.
4. Весь код приложения должен быть хорошо структурирован и комментирован.
5. Приложение должно быть отзывчивым и иметь хороший пользовательский интерфейс.

Дополнительные требования:

1. Приложение должно иметь возможность сортировки задач по различным критериям, например, по дате создания или статусу выполнения.
2. Приложение должно иметь возможность поиска задач по ключевым словам или фразам.
3. Приложение должно иметь возможность фильтрации задач по различным критериям, например, по статусу выполнения или приоритету.

Примечания:

1. Дизайн и структура приложения остаются на усмотрение кандидата. Мы заинтересованы в оценке личного подхода к разработке и владении соответствующими навыками.
2. Проект должен быть предоставлен в виде репозитория Git, к которому можно получить доступ.
